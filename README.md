# Sistema Unideportes

## Instalación de la Base de Datos
1. Abre **phpMyAdmin**.
2. Crea una base de datos llamada `"unideportes"`.
3. Importa el archivo situado en `/db/unideportes.sql`.

## Estructura Principal del CRUD
- **Create**: `insert_product.php`
- **Read**: `inventario.php` (Incluye buscador JS)
- **Update**: `ventas.php`
- **Delete**: `eliminar.php`
